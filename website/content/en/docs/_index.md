# Documentation

Welcome to the dvcx documentation. This guide will help you get started with dvcx and learn how to use all its features.

## What is dvcx?

dvcx (DevCycle CLI Extended) is an unofficial command-line tool for interacting with the DevCycle Management API. It provides functionality that is not available or limited in the official DevCycle CLI.
