# ドキュメント

dvcxドキュメントへようこそ。このガイドでは、dvcxの使い始め方とすべての機能の使い方を説明します。

## dvcxとは？

dvcx（DevCycle CLI Extended）は、DevCycle Management APIと対話するための非公式コマンドラインツールです。公式DevCycle CLIでは利用できない、または制限されている機能を提供します。
